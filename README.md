# SQL Practice Repository

## Overview

This repository contains my SQL practice work as I learn and improve my database querying and data analysis skills. It includes exercises starting from **basic SQL concepts** and gradually progressing to **intermediate-level queries and problem solving**.

The goal of this repository is to document my learning journey while building practical SQL skills that are useful in data analysis, data science, and database management.

## Topics Covered

The exercises in this repository cover a range of SQL concepts, including:

### Basic SQL

* SELECT statements
* Filtering data using WHERE
* Sorting results using ORDER BY
* Limiting results
* Basic aggregate functions (COUNT, SUM, AVG, MIN, MAX)

### Intermediate SQL

* GROUP BY and HAVING
* Joins (INNER JOIN, LEFT JOIN, RIGHT JOIN)
* Subqueries
* CASE statements
* String and date functions
* Data cleaning and transformation

## Practice Approach

Most files in this repository contain SQL queries that answer specific questions from datasets. These exercises help practice:

* Combining data from multiple tables
